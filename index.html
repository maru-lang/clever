🧠 Exercise 4: Grammar Spotlight: The "Because" & "So" Chain Reaction!

In a chain reaction, one thing causes another thing to happen (the effect).
We use these two magic words to link them, but they work in opposite directions!

Use BECAUSE to give the REASON (Why did it happen?)

[ Effect ] ← BECAUSE ← [ Cause ]

Example: The cat decided to hide because it was afraid.

Use SO to give the RESULT (What happened next?)

[ Cause ] → SO → [ Effect ]

Example: The cat was afraid, so it decided to hide.

Exercise 5: Build a Sentence (Same Story, New Structure!)

Let's practice the same story about Tariq, the kitten, and the dog. Finish each sentence logically. Notice how switching between "because" and "so" changes how you finish the thought!

  1. hide: The tiny kitten tried to hide at the bottom of the box because
  2. afraid: The kitten was very afraid, so
  3. attack: The big stray dog wanted to attack the box because
  4. clever: Tariq had a clever idea to use food, so
  5. agree: His mother finally had to agree to keep the kitten because

💡 Grammar Tip

Sentence 1 uses "because", so you must explain why it hid. Sentence 2 uses "so", so you must explain what happened next.

📝 Suggested Answers

  • ...because it was very afraid of the dog. (Reason)
  • ...so it tried to hide at the bottom of the box. (Result)
  • ...because it felt hungry and angry. (Reason)
  • ...so the angry dog stopped attacking. (Result)
  • ...because it was too cruel to leave the kitten outside. (Reason)
